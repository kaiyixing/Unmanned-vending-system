# 数据层本地部署指南

本作业数据层包含三个组件：**MySQL**（主数据库）、**Redis**（缓存/会话）、**MinIO**（图片存储）。

推荐用 **Docker Desktop** 一键部署，5 分钟即可全部跑起来。

---

## 方案一：Docker 部署（强烈推荐）

### 第一步：安装 Docker Desktop

1. 访问 https://www.docker.com/products/docker-desktop 下载 Windows 版
2. 安装过程中如果提示开启 WSL2，勾选并继续（Windows 11 通常已自带）
3. 安装完成后重启电脑，打开 Docker Desktop 确保左下角显示 **绿色 running** 状态

### 第二步：启动三大服务

在当前文件夹（有 `docker-compose.yml` 的位置）打开 PowerShell 或 Git Bash，执行：

```bash
docker-compose up -d
```

等待 1-2 分钟，三个容器会自动下载镜像并启动。

### 第三步：验证是否启动成功

```bash
docker ps
```

应该看到 `vending_mysql`、`vending_redis`、`vending_minio` 三个容器都在运行。

### 各服务连接信息

| 服务 | 地址 | 端口 | 用户名 | 密码 |
|-----|------|------|--------|------|
| MySQL | localhost | 3306 | root / vending | root1234 / vending1234 |
| Redis | localhost | 6379 | 无 | 无 |
| MinIO API | localhost | 9000 | minioadmin | minioadmin1234 |
| MinIO 控制台 | http://localhost:9001 | 9001 | minioadmin | minioadmin1234 |

### 初始化数据库

```bash
# 执行建表脚本
docker exec -i vending_mysql mysql -u vending -pvending1234 vending_db < vending-server/sql/schema.sql

# 导入初始数据
docker exec -i vending_mysql mysql -u vending -pvending1234 vending_db < vending-server/sql/init_data.sql
```

### Spring Boot 配置参考（application.yml）

项目已支持通过环境变量配置，避免硬编码敏感信息：

```yaml
server:
  port: ${SERVER_PORT:8080}

spring:
  datasource:
    url: ${DB_URL:jdbc:mysql://localhost:3306/vending_db?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai}
    username: ${DB_USERNAME:vending}
    password: ${DB_PASSWORD:vending1234}
    driver-class-name: com.mysql.cj.jdbc.Driver
  data:
    redis:
      host: ${REDIS_HOST:localhost}
      port: ${REDIS_PORT:6379}
      database: ${REDIS_DATABASE:0}

jwt:
  secret: ${JWT_SECRET:vending-system-secret-key-2026-must-be-at-least-256-bits}
  expiration: ${JWT_EXPIRATION:7200000}

minio:
  endpoint: ${MINIO_ENDPOINT:http://localhost:9000}
  access-key: ${MINIO_ACCESS_KEY:minioadmin}
  secret-key: ${MINIO_SECRET_KEY:minioadmin1234}
  bucket-name: ${MINIO_BUCKET:vending-images}
```

### 环境变量配置说明

生产环境强烈建议使用环境变量配置敏感信息：

| 环境变量 | 说明 | 默认值 |
|---------|------|--------|
| `SERVER_PORT` | 后端服务端口 | 8080 |
| `DB_URL` | 数据库连接 URL | `jdbc:mysql://localhost:3306/vending_db?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai&useSSL=false&allowPublicKeyRetrieval=true` |
| `DB_USERNAME` | 数据库用户名 | vending |
| `DB_PASSWORD` | 数据库密码 | vending1234 |
| `REDIS_HOST` | Redis 主机 | localhost |
| `REDIS_PORT` | Redis 端口 | 6379 |
| `REDIS_DATABASE` | Redis 数据库 | 0 |
| `JWT_SECRET` | JWT 密钥（生产环境请设置强密码） | 内置默认值 |
| `JWT_EXPIRATION` | JWT 过期时间（毫秒） | 7200000 |
| `MINIO_ENDPOINT` | MinIO 地址 | `http://localhost:9000` |
| `MINIO_ACCESS_KEY` | MinIO 访问密钥 | minioadmin |
| `MINIO_SECRET_KEY` | MinIO 密钥 | minioadmin1234 |
| `MINIO_BUCKET` | MinIO 存储桶 | vending-images |
| `LOG_LEVEL` | 日志级别 | info |
| `SECURITY_LOG_LEVEL` | 安全日志级别 | info |

#### Windows 设置环境变量（PowerShell）
```powershell
# 临时设置（当前终端有效）
$env:DB_PASSWORD="your_secure_password"
$env:JWT_SECRET="your_secure_jwt_secret"
$env:MINIO_SECRET_KEY="your_minio_secret"

# 然后启动服务
cd vending-server
mvn spring-boot:run
```

#### Linux/Mac 设置环境变量
```bash
# 临时设置（当前终端有效）
export DB_PASSWORD="your_secure_password"
export JWT_SECRET="your_secure_jwt_secret"
export MINIO_SECRET_KEY="your_minio_secret"

# 然后启动服务
cd vending-server
mvn spring-boot:run
```

#### 使用 .env 文件（推荐）

创建 `vending-server/.env` 文件：
```env
DB_PASSWORD=your_secure_password
JWT_SECRET=your_secure_jwt_secret
MINIO_SECRET_KEY=your_minio_secret
```

然后在启动时加载环境变量（需要配合相应的插件或工具）。

### 常用 Docker 命令

```bash
# 查看运行状态
docker-compose ps

# 查看日志（排错用）
docker-compose logs -f mysql
docker-compose logs -f redis
docker-compose logs -f minio

# 停止所有服务
docker-compose down

# 彻底删除（包括数据）
docker-compose down -v

# 重启单个服务
docker-compose restart mysql
```

## 方案二：分别安装（不用 Docker）

如果你电脑配置较低跑不动 Docker，可以分别安装：

### MySQL

1. 下载 MySQL Installer：https://dev.mysql.com/downloads/installer/
2. 选择 **Server Only** 安装，记住设置的 root 密码
3. 用 MySQL Workbench 或命令行创建数据库：
```sql
CREATE DATABASE vending_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'vending'@'localhost' IDENTIFIED BY 'vending1234';
GRANT ALL PRIVILEGES ON vending_db.* TO 'vending'@'localhost';
FLUSH PRIVILEGES;
```
4. 执行建表和初始化脚本

### Redis

1. 下载 Windows 版 Redis：https://github.com/tporadowski/redis/releases
2. 解压后双击 `redis-server.exe` 启动
3. 默认端口 6379，无密码

### MinIO

1. 下载 Windows 版：https://min.io/download#/windows
2. 在 PowerShell 中运行：
   ```powershell
   .\minio.exe server D:\minio-data --console-address ":9001"
   ```
3. 默认账号密码都是 `minioadmin`
4. 登录控制台创建 `vending-images` bucket

## 大作业简化建议

如果只是为了完成作业、跑通演示，**其实可以只装 MySQL**：
- Redis 可以用 Spring Boot 自带的 **Caffeine 本地缓存** 替代（不用装 Redis）
- 图片存储可以直接存 **本地磁盘** 或 **Base64 存数据库**（不用装 MinIO）

这样你本地只需要一个 MySQL，项目结构更简单，答辩演示也更稳定。

如果你确定要装全套，建议用 **方案一 Docker**，将来删除也干净，不会污染系统环境。

---

## 安全注意事项

⚠️ **重要提示**：
1. 生产环境务必修改所有默认密码
2. JWT_SECRET 必须设置为足够复杂的字符串（至少32字符）
3. 不要将包含敏感信息的配置文件提交到 Git
4. 建议使用 .gitignore 忽略 .env 文件
